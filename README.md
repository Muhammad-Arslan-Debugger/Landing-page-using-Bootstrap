Developer Blog Project

A responsive, multi-page blog website for coding enthusiasts built using **HTML5** and **Bootstrap 4**. This project features a modern layout with a navigation bar, interactive modals for authentication, a carousel for highlighted content, and stylized featurette sections.

## 🚀 Features

* **Responsive Design**: Fully responsive layout that works on mobile, tablet, and desktop screens.
* **Authentication Modals**: Interactive "Login" and "Sign Up" modals on the homepage.
* **Dynamic Navigation**: A dark-themed navbar with active state tracking and dropdown menus for topics like Technology and Web Development.
* **Modern Components**:
    * **Carousel**: A crossfade carousel to showcase major announcements.
    * **Featurettes**: Alternating image and text sections on the 'About' page.
    * **Cards**: Shadowed, hover-effect cards for blog posts with tags (World, Design, FrontEnd, Python).
* **Contact Form**: A comprehensive contact page including email input, query selection, and multi-check options (Member/Professor/Coder).

## 📂 Project Structure

* `index.html`: The main landing page featuring the blog feed and authentication.
* `about.html`: Information about the project's history and mission.
* `contact.html`: A functional contact form for user inquiries.

## 🛠️ Technology Stack

* **Frontend**: HTML5, CSS3
* **Framework**: Bootstrap 4.4.1
* **Interactivity**: jQuery 3.4.1, Popper.js

## 💻 How to Run

1. Clone or download the repository.
2. Ensure you have an active internet connection (required to load Bootstrap and jQuery from CDNs).
3. Open `index.html` in any modern web browser.
